# Security policy

Please **do not** file public GitHub issues for undisclosed security vulnerabilities.

Report them through **[GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories)** for this repository (maintainers will triage privately). Do not paste secrets, production tokens, or live customer data in any public channel.

For general security-hardening questions that are not sensitive vulnerability reports, you may open a normal public issue or discussion as appropriate.
